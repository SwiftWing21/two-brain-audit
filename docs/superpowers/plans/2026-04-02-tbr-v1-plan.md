- [ ] **Step 4: Create budget.py — per-session cost cap**

```python
class BudgetGuard:
    def __init__(self, max_usd: float = 1.0):
        self.max_usd = max_usd
        self.spent_usd = 0.0
    
    def check(self, estimated_cost: float) -> bool:
        """Return True if budget allows this call."""
        return (self.spent_usd + estimated_cost) <= self.max_usd
    
    def record(self, cost: float):
        self.spent_usd += cost
```

Wire into consensus_review and swarm_review — skip providers if budget exceeded.
